## 📖 About This Repository

**Git & GitHub Zero to Hero** is a structured, beginner-friendly learning repository that takes you from understanding what version control is, all the way through mastering Git internals, branching strategies, and real-world GitHub collaboration workflows.

Every topic has its own dedicated folder with a `README.md` containing in-depth explanations, command references, diagrams, and practical examples.

By the end of this course, you will be able to:

- ✅ Understand how Git works internally — the three areas, objects, and data flow
- ✅ Confidently use all essential Git commands for day-to-day development
- ✅ Create, switch, merge, and manage branches like a professional
- ✅ Resolve merge conflicts and understand fast-forward vs three-way merges
- ✅ Use Git Stash to manage work-in-progress safely
- ✅ Undo changes safely using reset and revert — know which to use and when
- ✅ Understand GitHub's full ecosystem — Actions, GHCR, Projects, Codespaces
- ✅ Contribute to open-source using the Fork → Clone → PR workflow
- ✅ Authenticate to GitHub securely using SSH and Personal Access Tokens

## 🛠️ Prerequisites

Before starting, make sure you have:

- A computer running Linux, macOS, or Windows
- Basic familiarity with the terminal / command line
- A free [GitHub account](https://github.com/signup)

## 🚦 Getting Started

```bash
# Clone this repository
git clone https://github.com/your-username/Git-GitHub-Zero-to-Hero.git

# Navigate into the project
cd Git-GitHub-Zero-to-Hero

# Start with the introduction
cd what-is-vcs
```

Work through each folder in order for the best learning experience. Each `README.md` builds on the previous topics.
